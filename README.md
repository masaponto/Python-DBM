# Python-DBM (DBM-MLP)
Decision Boundary Making (DBM) using MLP implemented in python3   


## How to install 
I recommend you to install this to anaconda3-4.0.0  

```
pip install git+https://github.com/masaponto/python-dbm 
```

## Require
- scikit-learn: 0.17.1
- numpy: 1.10.4
